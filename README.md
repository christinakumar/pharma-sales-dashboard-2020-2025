# pharma-sales-dashboard-2020-2025

## Project Overview
This project presents an interactive Pharmaceutical Sales Dashboard developed using Excel and Power BI to analyze sales performance, inventory utilization, and business trends from 2020 to 2025. The dashboard provides key insights to support data-driven business decisions.

## Business Objective
To analyze pharmaceutical sales data, monitor revenue trends, identify unsold inventory, evaluate capital utilization, and provide actionable insights for improving business performance.

## Tools Used
- Microsoft Excel
  - Data cleaning
  - Pivot Tables
  - Revenue calculations
  - Data analysis
- Power BI
  - Data visualization
  - DAX measures
  - cards
  - Interactive Dashboard

## Data Source
The dataset used in this project is a sample pharmaceutical sales dataset created for learning and portfolio purposes.

## Key Performance Indicators (KPIs)
- Total Revenue
- Total Units Sold
- Unsold Stock Value
- Capital Utilisation (%)

## Dashboard Features
- Revenue Trend by Year (2020–2025)
- Revenue by Country
- Revenue by Product Category
- Unsold Stock Analysis
- Capital Utilisation KPI
- Interactive Filters and Slicers

## Key Insights
- Revenue showed a declining trend after 2021.
- Capital utilisation remained high at approximately 85%.
- Unsold inventory represented around 15% of total inventory value.
- Country-wise analysis highlighted variations in sales performance.
- Inventory analysis helped identify opportunities to reduce unsold stock.

## Recommendations
- Improve inventory planning for products with high unsold stock
